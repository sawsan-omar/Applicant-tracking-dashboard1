# Applicant Tracking System — Prototype

## Overview
A simple Applicant Tracking Dashboard prototype built with plain HTML, CSS, and JavaScript.
Data persistence uses Local Storage (no backend required). SQL scripts and ERD are included
to demonstrate how this prototype can scale to a real MySQL-backed system.

## Features
- Add applicant: Name, Email, City, Skills (comma-separated)
- Save applicants to Local Storage
- Display applicants in a table
- SQL file for MySQL table and sample queries
- ERD diagram (single-table design)

## Files
- `index.html` — Frontend page
- `style.css` — Styles
- `App.js` — JavaScript logic (Local Storage)
- `Data.sql` — MySQL table create & queries
- `report.pdf` — Project report
